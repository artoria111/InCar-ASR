## 2. 评估指标

| 指标 | 目标值 | 评估方法 |
|------|:--:|------|
| CER (安静) | < 3% | 无噪声条件下 |
| CER (噪声) | < 8% | SNR 5-15dB |
| RTF | < 0.1 | Ascend 310B 实测 |
| 端到端延迟 | < 500ms | 音频结束→文本输出 |
| VAD 准确率 | > 95% | 噪声环境下的语音/非语音分类 |

## 3. 测试流程

1. 准备测试音频: 干净语音 × 噪声混合 (augment_audio.py)
2. 批量推理: 01_e2e_system.py --mode batch
3. CER 统计: 按维度分组生成报告
4. Bad case 分析: 导出高错误率样本
5. 最终报告: 输出 Markdown + JSON
