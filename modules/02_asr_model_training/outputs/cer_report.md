## 4. Bad Case 分析

高错误率样本主要特征:
- **未登录词**: 人名、地名不在模型词表中
- **数字串**: 纯数字序列容易漏识
- **长句 (>20 字)**: 注意力跨度不足

车载指令场景不受以上因素影响。

## 5. 结论

1. Paraformer-large 在 AISHELL-1 上 CER 1.95% (文献) / 1.29% (实测)，**满足 < 3% 目标**。
2. Paraformer-tiny 适用于车载指令词识别，部署体积 25MB。
3. 车载噪声环境 CER 评估需成员A 噪声增强测试集 (SNR 5-15dB) 产出后补齐。
4. 板端性能指标基本达标 (RTF 0.105 接近目标 0.1)。
