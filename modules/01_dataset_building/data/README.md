# Audio dataset layout / 音频数据集结构

Audio files are deliberately excluded from normal Git history. Keep local files in the following structure.

音频文件不会进入普通 Git 历史。请按照以下结构保存本地数据。

```text
data/
├── raw/<category>/*.wav
├── processed/<split>/<category>/*.wav
├── metadata.csv
└── processed/metadata.csv
```

`raw/` contains immutable source recordings. Never overwrite or manually normalize these files. `processed/` contains reproducible output generated by `scripts/audio/preprocess_audio.py`.

`raw/` 保存不可变的原始录音，不得覆盖或手动归一化这些文件。`processed/` 保存由 `scripts/audio/preprocess_audio.py` 生成的可复现结果。

## Metadata rules / 元数据规则

Copy `metadata_template.csv` to `metadata.csv` when importing recordings that were not created by the collection script. One row represents one source recording.

导入并非由采集脚本生成的录音时，请将 `metadata_template.csv` 复制为 `metadata.csv`。每一行对应一段原始录音。

- `recording_id` must be unique and stable. / `recording_id` 必须唯一且保持稳定。
- `category` should match a category in `configs/audio.yaml`. / `category` 应与 `configs/audio.yaml` 中的类别一致。
- `source_type` should be `self_recorded` or `public_dataset`. / `source_type` 应填写 `self_recorded` 或 `public_dataset`。
- `source_url` and `license` are required for third-party data. / 第三方数据必须填写 `source_url` 和 `license`。
- Review `contains_speech` before sharing and do not publish identifiable conversations. / 分享前必须检查 `contains_speech`，不得公开可识别身份的对话。
- Do not store precise locations, plates, or personal details. / 不得保存精确位置、车牌或个人信息。

The preprocessing script writes segment-level metadata automatically. All segments from one source recording are assigned to one dataset split to prevent train/test leakage.

预处理脚本会自动写入片段级元数据。同一原始录音产生的所有片段只会分配到同一个数据集划分中，从而防止训练集与测试集之间的数据泄漏。

## Sharing the audio / 音频共享

Use the storage location agreed by the group and record its download and checksum instructions here. If Git LFS is adopted later, migrate only after the whole group agrees. Never commit large WAV files to standard Git history.

请使用小组约定的存储位置，并在这里记录下载方法和校验值。如果后续决定使用 Git LFS，应在全组达成一致后再迁移。不要将大型 WAV 文件提交到普通 Git 历史中。
