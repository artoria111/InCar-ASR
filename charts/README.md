# Archived presentation charts

These PNG files came from an earlier presentation draft. Several labels mix
targets, estimates, literature values, and local observations. They are kept
only as historical design material and are **not release or benchmark
evidence**.

Current measured results must be generated from the evaluation and Atlas
reporting commands documented in `docs/TRAINING_AND_EVALUATION.md` and
`docs/RELEASE.md`. The old chart generator was removed so unverified values
cannot be accidentally regenerated as if they were current results.
